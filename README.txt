# Recipe Book Project

Welcome to the Recipe Book project! This is a web application that allows users to store and manage their favorite recipes. With this app, you can add, edit, and delete recipes, as well as view random recipe suggestions from an external API.

## Features

- Add new recipes with name, category, ingredients, and instructions.
- Edit existing recipes to update their details.
- Delete unwanted recipes from the collection.
- Get a random recipe suggestion with just a click.
- Enjoy a modern and trendy user interface design.

## Technologies Used

- HTML, CSS, and JavaScript for front-end development.
- Fetch API to retrieve random recipe data from an external source.
- Git for version control and collaboration.
- GitHub for hosting and managing the project.

## How to Use

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser.
3. Start managing your recipes and exploring new culinary delights!

Feel free to contribute, report issues, or suggest improvements to make this Recipe Book even better!
